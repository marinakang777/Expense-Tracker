## Описание программы

Expense Tracker — это простое десктопное приложение на Python для отслеживания личных финансов. Позволяет добавлять расходы, фильтровать их по категории и дате, а также считать сумму расходов за произвольный период. Данные сохраняются в файл JSON.

## Как использовать

1. Убедитесь, что у вас установлен Python 3.6+.
2. Запустите файл:
   ```bash
   python expense_tracker.py
3. Введите сумму, выберите категорию и укажите дату в формате ГГГГ-ММ-ДД.

4. Нажмите «Добавить расход» — запись появится в таблице.

5. Для фильтрации выберите категорию и/или диапазон дат, затем нажмите «Применить фильтр».

6. Чтобы посчитать сумму за период, укажите даты в полях фильтра и нажмите «Подсчитать сумму за период».

7. Данные автоматически сохраняются в expenses.json.

Примеры использования
Пример 1: Добавление расходов
Сумма: 250.50

Категория: Еда

Дата: 2026-05-19
→ Нажать «Добавить расход»

Пример 2: Фильтр по категории
Выбрать категорию «Транспорт» → нажать «Применить фильтр» → в таблице останутся только расходы на транспорт.

Пример 3: Подсчёт суммы за месяц
В поле «Дата от» ввести 2026-05-01, в поле «Дата до» — `2026-05-31», нажать «Подсчитать сумму за период».

Тестирование (возможные проверки)

Ввод отрицательной суммы	Ошибка «Сумма должна быть положительной»
Ввод текста в поле суммы	Ошибка «Некорректная сумма»
Неверный формат даты	        Ошибка «Неверный формат даты»
Фильтр без данных	        Таблица пуста, статус показывает 0 записей
Сумма за период без расходов	Выводится 0.00 руб.


## 🧪 Проверка корректности ввода

- **Сумма**: проверяется на `float` и положительное значение.
- **Дата**: строго по шаблону `YYYY-MM-DD`.
- Пустые поля категории или даты не допускаются (есть проверки).
